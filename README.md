# RecentNewsSkill 

* simple alexa skill feed news use newsapi.org .
* based on ask-cli project.
* no use account link, only use user session.

## prepare
* setup ask-cli.
* make api_key for newsapi.org with KMS and lambda env.
* make dynamoDB table.

## deploy
```
ask deploy -p ${profile}
```
